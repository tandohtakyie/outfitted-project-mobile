Bodoni Moda Variable Font
=========================

This download contains Bodoni Moda as both variable fonts and static fonts.

Bodoni Moda is a variable font with these axes:
  opsz
  wght

This means all the styles are contained in these files:
  BodoniModa-VariableFont_opsz,wght.ttf
  BodoniModa-Italic-VariableFont_opsz,wght.ttf

Get started
-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles

-------------------------------

  https://developers.google.com/web/fundamentals/design-and-ux/typography/variable-fonts
  https://variablefonts.typenetwork.com
  https://medium.com/variable-fonts
